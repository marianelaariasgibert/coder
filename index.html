1- definir nombre de usuario: git config --global user.name "Marianela" 2- definir email: git config --global user.email maru@gmail.com 3- iniciar repositorio: git init 4- ver el estado de mis archivos: git status 5- debo llevarlos a staging area: git add . (el punto engloba todo o puedo poner el nombre) 6- luego a commit: git commit -m "texto referente a commit" 7- vemos los comandos del commit: git log. una sola linea con git log --oneline 8- para volver a la version anterior: git checkout y el numero de commit 9- para cambiar de rama: git checkout nomnbre de rama 10- para fusionar las versiones: git merge nombre de la rama 11- para ver las ramas: git branch 12- crear ramas: git branch nombrerama 13- cambiar de rama: git checkout nombrerama 14- para guardar los cambios hechos desde la pagina de github usamos: git pull 15- enlazar repositori: git remote add origin https://github.com/marianelaariasgibert/coder.git 16- git push -u origen nombredelarama
