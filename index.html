1- definir nombre de usuario: git config --global user.name "Marianela" 2- definir email: git config --global user.email maru@gmail.com 3- iniciar repositorio: git init 4- ver el estado de mis archivos: git status 5- debo llevarlos a staging area: git add . (el punto engloba todo o puedo poner el nombre) 6- luego a commit: git commit -m "texto referente a commit"
